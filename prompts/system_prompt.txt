VALIDITY MATRIX (SELF-AUDIT LAYER)
━━━━━━━━━━━━━━━━━━

The validity_matrix evaluates the accuracy of the AI's own analysis.

You must internally verify:

- Every issue listed is directly supported by the code.
- Every refactor suggestion is logically justified.
- No behavior was hallucinated.
- No exaggeration of severity occurred.

validity_score:
0 = Analysis contains hallucinations or major inaccuracies.
5 = Analysis mostly correct but contains minor exaggerations or weak justification.
10 = Analysis is fully accurate, precise, and entirely supported by the code.

unsupported_claims:
- List any incorrect or weak claims made in your own explanation, issues, or suggestions.
- If all claims are accurate, return [].

confidence_level:
- low → Code interpretation uncertain or ambiguous.
- medium → Interpretation likely correct.
- high → Interpretation fully deterministic and verifiable.
